Nuestros espacios

    RECEPCIÓN

En este espacio vas a poder realizar cualquier tramite administrativo o consulta que tengas en relación a tu plan vigente. También vas a poder reservar turno para el apto médico o para nuestro nutricionista.

    PESO LIBRE

En nuestro salón de peso libre vas a encontrar bancos para realizar trabajo de pectorales, poleas y todo nuestro inventario de mancuernas. Es un espacio perfecto para trabajar ejercicios que no requieran de máquinas.

    SALÓN DE AERÓBICA

El salón de aeróbica se encuentra en el primer piso y vas a poder encontrar todas nuestras máquinas para realizar un trabajo aeróbico. Contamos con bicicletas fijas, cintas y elípticos.

    TULUKA

Este salón se encuentra en el tercer piso y aqui mismo vas a poder realizar tu entrenamiento de Tuluka. Contamos con barras olímpicas, discos, cajones, kettlebell, mancuernas y todo el material que necesitas para tus rutinas de entrenamiento.

    MARKET NUTRITIVO

En nuestro Market vas a poder encontrar productos saludables y suplementación para tu entrenamiento. Vas a recibir asesoramiento por parte de nuestros especialistas, para poder obtener los mejores resultados. También podrás usar este espacio para trabajar o pasar un lindo rato.

    MUSCULACIÓN

Nuestro salón de musculación vas a poder encontrar todas nuestras máquinas y diferentes tipos de materiales para adaptar a tus rutinas. Este espacio es ideal para reaizar trabajos con máquinas y todo tipo de circuitos.

    SALÓN FITNESS

Este salón esta dedicado exclusivamente para todas nuestras clases. Vas a poder realizar diferentes tipos de clases, como por ejemplo Zumba, Body Pump, Power Jump, Localizada, etc.

    SPORT CYCLE

Espacio dedicado exclusivamente para nuestras clases de Sport Cycle. También vas a poder venir a este esacio para usar nuestras bicicletas fijas de Spinning en cualquier momento del día.
